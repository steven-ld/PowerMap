反向映射（把本机服务暴露给内网）

反向映射让 server 端在其内网监听，把连接经隧道交回本机，由本机拨号本地或家庭网络里的目标。 这会让持有 server 的一方触达你这台机器上的服务，因此默认全部拒绝：必须显式填写允许的网段与端口才会放行。 具体在内网监听哪些地址，由 server 端的 reverse 配置决定。

近期事件
